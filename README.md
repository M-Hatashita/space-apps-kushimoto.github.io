## 開発者向け

```
$ git clone git@github.com:space-apps-kushimoto/space-apps-kushimoto.git
$ cd space-apps-kushimot
$ bundle install --path vendor
```

ローカル環境の起動

```
$ bundle exec jekyll serve
```
